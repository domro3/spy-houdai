# Public Alpha v0.1 Feedback Plan

作成日: 2026-07-01

ステータス: M6 AI代替チェックでは手動収集に決定

## 目的

Public Alpha v0.1では、機能の多さではなく、初見入口と1ゲームの手触りを確認する。

## 送信先

Public Alpha v0.1の初回公開では、ゲーム内フォームや外部投稿フォームを持たない。

配布時にこの質問リストを添えて、回答は手動で受け取り、この文書または別途作成するフィードバックログへ転記する。

理由:

- Git remoteが未設定で、GitHub Issues / Discussionsを公開導線にできない
- Google Formsなど外部フォームは、このリポジトリだけでは作成・公開確認できない
- Public Alpha v0.1では、まず少人数の手動フィードバックで十分

## 最低限の質問

1. 何をするゲームか、最初の画面で分かりましたか。
2. 次に押す操作で迷いましたか。
3. ボスHP、拠点耐久、ボス予告は読めましたか。
4. `撃つ`、`守る`、`直す` の使い分けは分かりましたか。
5. スパイ妨害や通信ノイズに気づきましたか。
6. 結果画面で勝者、スパイ正体、称号は読めましたか。
7. もう一戦したいと思いましたか。
8. スマホで文字やボタンが小さい箇所はありましたか。
9. 途中で止まった、押せない、表示が崩れた箇所はありましたか。
10. 一番よかった点と、一番直したい点を教えてください。

## 記録フィールド

| 項目 | 形式 | メモ |
| --- | --- | --- |
| プレイ日 | 日付 | 回答者入力または自動 |
| 端末 | 選択式 | iPhone / Android / PC / その他 |
| ブラウザ | 選択式 | Safari / Chrome / Edge / その他 |
| 画面幅 | 任意 | 分かる場合だけ |
| 完走したか | はい/いいえ | 結果画面まで到達したか |
| 迷った箇所 | 自由記述 | UI改善用 |
| 楽しかった箇所 | 自由記述 | 強み確認用 |
| もう一戦感 | 1から5 | 継続意欲 |
| 致命的不具合 | 自由記述 | 公開停止判断用 |

## 公開後の判断

| 条件 | 対応 |
| --- | --- |
| 進行不能報告が複数出る | 公開停止またはHotfix |
| スマホで押せない報告が多い | UI優先修正 |
| 目的が伝わらない報告が多い | 入口コピーと最初の行動説明を修正 |
| 妨害に気づかない報告が多い | 通信ノイズ演出をM6/M7で強化 |
| もう一戦感が低い | 結果演出、称号、ゲーム時間を再調整 |
