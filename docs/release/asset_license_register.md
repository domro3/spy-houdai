# Asset And License Register

作成日: 2026-07-01

用途: Public Alpha v0.1公開前の素材・ライセンス確認台帳。

## 現在の公開ビルド方針

Public Alpha v0.1では、出所未記録の画像、BGM、SE、外部フォントを配布しない。

画面上のボス、リンクコア、砲台、エフェクトは、コード内のCSS/SVGベースの仮ビジュアルを使う。

## 素材一覧

| 種別 | 現状 | 公開ビルドでの扱い | 確認状況 |
| --- | --- | --- | --- |
| ボス/砲台/リンクコア | Reactコンポーネント内SVG/CSS | 使用する | 自作コードとして扱う |
| UIアイコン | `lucide-react` | 使用する | `node_modules/lucide-react/package.json` で ISC license を確認 |
| フォント | システムフォント | 使用する | 外部フォントなし |
| BGM | 未使用 | 配布しない | 追加時に別途確認 |
| SE | 未使用 | 配布しない | 追加時に別途確認 |
| 生成AI画像 | 公開ビルドでは未使用 | 配布しない | 追加時に生成記録を残す |
| プロトタイプPNG | 旧M4検討用 | `public/` から削除 | 出所記録がないため配布対象外 |

## Lucide確認

ローカル確認:

```text
package: lucide-react
version: 0.562.0
license: ISC
```

公開ページやREADMEに第三者素材のクレジット欄を作る場合、`lucide-react` の利用を記録する。

## 名称・商標予備確認

2026-07-01時点で、AIによる一般Web完全一致検索と、JPO/J-PlatPatでの公式確認手順を整理した。

これは法律判断ではなく、公開前に人間が公式検索で確認すべき対象を絞るための予備記録である。

- 公式確認先: https://www.j-platpat.inpit.go.jp/
- JPOの検索案内: https://www.jpo.go.jp/support/startup/shohyo_search.html
- 公式確認では、J-PlatPatの商標検索および称呼類似検索を使う。

| 対象 | 予備検索メモ | M6での扱い |
| --- | --- | --- |
| スパイ砲台 | 一般Webで「スパイ砲台シリーズ」の既存タグを確認。商標ではなく動画タグだが、完全な未使用語ではない。 | Public Alphaでは仮名扱い。正式タイトル化前にJ-PlatPat確認 |
| スパイ放題 | 一般Webでは普通表現としての利用が見える。ゲームタイトルとしての強い一致は未確認。 | タイトル候補としては優先しない |
| SPY HOUDAI / Spy Houdai | 一般Web完全一致では強い既存タイトルを確認できず。 | 英字表記は正式化前にJ-PlatPat確認 |
| Spy Turret | Fortnite関連のコンセプト文脈などで既存利用あり。 | 公開タイトルや固有名にはしない |
| みんな砲台 | ゲーム実況・攻略文脈などで一般表現として既存利用あり。 | コピー候補としては弱める |
| 怪しいコイン | 外部ゲームイベントや一般表現として既存利用あり。 | 主要アイテム名としては代替名も検討 |
| オーバードライブ作戦 | 一般的な攻略・ファン文脈で既存利用あり。 | 内部作戦名に留め、公開コピーでは前面に出さない |
| 緊急弱体化プロトコル | 一般Web完全一致で強い既存タイトルは未確認。 | 公式検索待ち |
