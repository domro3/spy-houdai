# Alpha Test Summary 2026-06-29

対象: M4 Post-M4人間プレイテスト

記録元: `docs/playtest/alpha_checklist.md`

## 判定

ステータス: M4 Alpha Playtest Ready

1ゲームの人間チェックは成立した。M4はAI Alpha Readyに加えて、人間スモークチェックでもAlpha Playtest Readyと見なせる。

## 確認できたこと

- `/board`、`/player/p1` から `/player/p5`、`/debug` のセットアップは完了した
- 全プレイヤー端末がリンクし、Boardに5基分の同期状態が出た
- Boardに進行中のスパイ正体、スパイ専用行動名、個別ログ、debugLogは出ていない
- Player端末では本人の役職だけが表示され、スパイ用行動はスパイ端末だけに出た
- 守る、直す、大技チャージ、狙い撃ち、拠点耐久危険は意図として伝わった
- ログを読みすぎずに遊べた
- Boardだけでも戦況は分かった
- 1ゲームが最後まで進み、5から8分目安に近く、GMなしでも進行が止まらなかった
- スパイ本人は自分がスパイだと迷わなかった
- 砲台チームがBoardからスパイ情報を見抜いてしまうことはなかった
- もう一度テストしてよい状態と判断された

## 見つかった課題

- Boardの「プレイヤー状態カード」という表現、または該当UIの意図が分かりにくい
- スパイ妨害に気づきにくい
- Player端末だけでは次に押す操作が分かりにくい
- 作戦端末で迷いが出た
- 「個別状態」という文言が分かりにくい
- 他プレイヤーの選択待ち時間が退屈に感じられる
- UIのワクワク感が不足している

## 次の改善候補

1. Player端末の次操作表示を強める
2. 「個別状態」を、プレイヤーに伝わる言葉へ変更する
3. スパイ妨害の気づきやすさを上げる
4. 他プレイヤー待ち時間に、待っている理由や同期状況の動きを出す
5. Boardのプレイヤー状態カードを、何を見る場所なのか分かる見出しへ整理する
6. UIのワクワク感を増やす演出候補をM5へ送る
