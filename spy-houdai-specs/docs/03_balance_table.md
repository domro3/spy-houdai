# スパイ砲台 バランス調整表 v0.1

## 1. 基本方針

『スパイ砲台』はガチ競技ゲームではなく、楽しさ重視のパーティゲームである。

理想目安：

| 項目 | 目安 |
|---|---:|
| 砲台チーム勝利率 | 55〜65% |
| スパイ勝利率 | 35〜45% |
| スパイ裏勝利発生率 | 20〜35% |
| 1ゲーム中の明確な異常 | 2〜4回 |
| ボス撃破が最終ラウンドまでもつれる割合 | 高め |

## 2. プレイヤー人数別基本値

| 人数 | ボスHP | 拠点耐久 | ラウンド数 | スパイ人数 |
|---:|---:|---:|---:|---:|
| 4人 | 430 | 100 | 5 | 1 |
| 5人 | 560 | 100 | 5 | 1 |
| 6人 | 710 | 100 | 5 | 1 |

## 3. ボス側基本値

| 項目 | 初期値 |
|---|---:|
| 通常ボス攻撃 | 10 |

| 弱攻撃 | 10 |

| 拠点耐久初期値 | 100 |
| ボス特殊行動発生率 | 20% |

## 4. 砲台チーム行動

| 行動 | 主な値 | 調整意図 |
|---|---|---|
| 通常攻撃 | 60ダメージ | 安定行動 |
| チャージ攻撃 | 成功100 / 失敗30 / 成功75% | 高火力だが疑われる |
| 防御 | ボス攻撃50%軽減 | 拠点を守る |
| 修理 | 20回復 | 拠点耐久を回復 |
| スキャン | 曖昧なヒント | 疑惑を動かす |

## 5. スパイ行動

| 行動 | 主な値 | 調整意図 |
|---|---|---|
| 偽装攻撃 | 30ダメージ | 攻撃しているふり |
| ボス回復 | 50回復 | 強いが目立つ |
| 妨害 | 攻撃30%減 / チャージ成功率-20% | 中心妨害 |
| ログ撹乱 | ログを曖昧化 | 推理材料を減らすが目立つ |
| 怪しいコイン | ラウンド3以降 / 40%で2票 | 投票救済 |

## 7. 中間作戦分岐

### オーバードライブ作戦

| 項目 | 値 |
|---|---:|
| ボス攻撃力 | +5〜10 |
| ボス防御 | 少し上昇 |
| 特殊行動 | 追加 |
| 成功時評価 | 大幅上昇 |

具体効果案：

- ラウンド4〜5のボス攻撃+10
- ボスの受けるダメージ10%軽減
- 成功時スコア倍率+1.3
- 称号「オーバードライブ突破」

### 緊急弱体化プロトコル

| 項目 | 値 |
|---|---:|
| ボス攻撃力 | -5〜10 |
| ボス防御 | 低下 |
| 特殊行動 | 一部封印 |
| 成功時評価 | 控えめ |

具体効果案：

- ラウンド4〜5のボス攻撃-5
- ボスの受けるダメージ10%増加
- 最終評価上限をAまでに制限
- 「緊急支援使用」ログを結果に表示

## 8. 勝敗・評価

| 条件 | 結果 |
|---|---|
| ボスHPを0にする | 砲台チーム勝利 |
| ラウンド5終了時点でボスHPが残る | スパイ勝利 |
| 拠点耐久が0になる | スパイ勝利 |
| ボス撃破後、スパイを当てられない | スパイ裏勝利 |

| 称号 | 判定基準 |

| MVP | 総貢献値最大 |

| 最大回復 | 回復量最大 |
| 最大防御 | 軽減量最大 |
